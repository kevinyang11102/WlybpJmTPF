# 民宿平台管理系统 java210

## 项目概述

民宿平台管理系统是基于SpringBoot和vue3技术构建的前后端分离系统。该系统支持普通游客（用户）和管理员两种角色，提供从民宿浏览、搜索、预订到后台管理的全方位功能。

## 技术栈

- 后端：SpringBoot
- 前端：vue3
- 数据库：MySQL

## 功能模块

### 前端功能

- **系统首页**：展示民宿酒店类型，提供时间搜索、民宿名称和民宿类型搜索功能。
- **房间简介**：查看民宿简介和评论。
- **预定民宿**：在民宿详情页面进行预订。

### 用户功能

- **用户登录/注册**：用户需登录后才能进行预订。

### 后台管理系统

#### 可视化统计模块

- 预订访问统计可视化界面

#### 用户管理模块

- 用户管理
- 管理员工管理
- 员工管理

#### 系统菜单管理模块

- 菜单管理
- 角色管理
- 日志管理
- 文件管理

#### 民宿管理模块

- 房源管理
- 房源图片管理

#### 订单管理模块

- 订单管理
- 财务管理

#### 业务办理模块

- 民宿预订/入住管理
- 预订处理
- 入住管理/退房管理

## 系统角色

- 普通游客（用户）
- 管理员
